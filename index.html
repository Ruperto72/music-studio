🎛️ Music Studio — Help

Overview

A browser tool for composing 8-bit chiptune loops — everything is synthesised with the Web Audio API, no audio files. Instruments are laid out as separate tracks stacked vertically, like a music studio (Pro Tools style). A shared timeline at the top and the red playhead run across every lane at once. The current project's name is shown at the top left (click it to rename) — open 🎵 in the Session group to load an example, one of your own saved songs, or start a new one; your own songs are stored in this browser.

Tracks & channel strips

  • Starts with a lane for each track in the loaded song (e.g. Lead, Harmony, Bass, …) plus a fixed Rhythm lane with six hit types: Kick, Snare, Hi-hat, Tom, Clap and Crash.
  • Each track's channel strip (left) has its name, Mute / Solo, a volume slider, waveform, pan and a level meter.
  • ＋ Add track (next to the Master bar at the bottom) adds a tonal track; double-click a track's name to rename it; the ✕ on a strip removes that track (Rhythm is fixed). Added tracks export into the game too.
  • The ▾/▸ button on a track collapses it to a slim strip — name, Mute/Solo and the level meter stay visible, everything else (waveform, automation/envelope, volume/pan, remove) hides along with the grid. Handy for an overview with many tracks. Remembered per browser.
  • The 〰 Automation button opens a curve editor for that track's Volume or Pan over time: click empty space to add a point, drag a point to move it, double-click to delete it. Without any points, the channel strip's slider value is used for the whole song (as before).
  • The E (Envelope) button opens that track's ADSR — Attack/Decay/Sustain/Release, shaping how a note's volume rises and fades. Attack/Decay/Release are a percentage of each note's own length (so they scale with short vs. long notes); Sustain is the held level. Reset restores the default shape.
  • Click a strip (or into its lane) to make it the active track that the tools edit.
  • Pitch lanes auto-fit to the notes they contain, so tracks stay compact.

Tools

  • ✏️ Pen — click an empty cell to add a note/hit.
  • 🧽 Eraser — click a note/hit to remove it (or a whole selected group).
  • ✋ Grab — drag a note/hit to move it; drag its right edge to change its length.
  • The Grid menu sets the snap resolution: 1/4, 1/8, 1/16, or triplets (1/8 T, 1/16 T). New notes take the grid length, and moving/resizing/nudging snaps to it — so you can place sixteenths and triplet figures between the eighth lines. Finer grids draw extra guide lines.
  • Placing or selecting a note/hit plays it, so you hear the pitch or drum as you edit.

Selecting several notes

  • Shift+click each note/hit to add or remove it from the selection (any tool).
  • With Grab, drag across empty space to marquee-select everything whose start falls in the range.
  • A selection moves together; Delete removes it; Ctrl/Cmd+C / V copies & pastes it at the playhead.

Note effects

Select a single note to edit vibrato, tremolo, bend, duty cycle, arpeggio, portamento, bitcrush, echo and chorus in the inspector on the right.

Bottom bar (Tempo · Length · Grid · Master)

The strip under the tracks holds Tempo, Length, Grid (the note snap resolution), the 🔊 Master volume and ＋ Add track. The ▾ button collapses it to a slim bar to free up space (remembered between sessions). Length makes the song longer/shorter one bar at a time: + appends an empty bar, − removes the last bar (dropping any notes/hits past the new end); the length is saved with your file and exported so the game loops it.

Mixing

  • Each track's volume slider sets its relative level (its mix) — drag while playing to rebalance live.
  • The waveform picker (Square / Triangle / Saw / Sine) and the pan slider set each track's tone and stereo placement. Double-click the pan slider to re-center it.
  • Mute silences a track; Solo silences every track except the soloed one(s).
  • 🔊 Master sets the overall preview loudness (editor only — not part of the game's audio).
  • The level meters (per track and next to Master) show live output while previewing or playing.

Transport & timeline

  • Transport: return-to-start, stop, play, and a loop on/off toggle; the LCD shows Bars | Beats and Min:Sec.
  • Click the timeline (or drag the playhead) to set where playback starts; the view auto-scrolls to follow the playhead.
  • The green band is the loop range — drag its ends, or "Full range" to reset it. ⧉ Repeat duplicates that range's notes/hits into the region right after it (growing the song if needed) and advances the range onto the copy, so clicking again keeps stamping it forward.
  • The Meter menu (bottom bar) sets the time signature (4/4, 3/4, 6/8, …), which regroups the bar lines, numbers and Length. Note times don't move.
  • 🚩 adds a marker at the playhead; click a marker flag to rename it (or clear the name to delete). Markers are saved with your file but aren't part of the game.

Saving & exporting

  • 🎵 opens the Song library: load an example, one of your saved songs, or start a new song — named up front, either with empty Melody/Bass/Rhythm starter tracks or a blank project (just Rhythm). The Song panel above the inspector shows the project name (click-to-rename any time) and, via its ▾, tempo/meter/length/track-count at a glance.
  • 💾 Save / 📂 Load keep a work-in-progress .json (name, tracks, tempo, length, meter, mix, waveforms, pan, automation, envelopes & markers). Work also autosaves to this browser.
  • ⤓ Export emits the TRACKS and RHYTHM_TRACK declarations to paste over the ones in audio.js; click it again to close the box.
  • ⛶ Fullscreen toggles the browser's fullscreen mode. On Android, your browser's menu also offers Add to Home screen / Install app — this page is installable and works offline once loaded.

🎚 Instrument presets

Saves this track's waveform + envelope (attack/decay/sustain/release) as a named preset you can apply to any tonal track, in any song. Saved in this browser (nothing is uploaded).

🥁 Rhythm patterns

Inserts a one-bar groove into the Rhythm track, tiled from the playhead's bar through the end of the song (replacing whatever was there). Move the playhead first if you only want to fill part of the song.
